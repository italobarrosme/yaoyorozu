# Style

## Pattern installation styles

Install `postcss-import` and `tailwindcss` in the project.

<Unstyled>
  <Copy language='bash'>
    {`
      npm install postcss-import tailwindcss
    `}
  </Copy>
</Unstyled>

Create the file `postcss.config.js` in the root of the project.

<Unstyled>
  <Copy language='yaml'>
    {`
      module.exports = {
        plugins: {
          'postcss-import': {},
          tailwindcss: {},
          autoprefixer: {},
        },
      }
    `}
  </Copy>
</Unstyled>

Create global styles for the colors and config tailwind in the file `**/global.css`.

<Unstyled>
  <Copy language='yaml'>
    {`
      @import '@developerskyi/react-components/dist/styles.css';
      @import 'tailwindcss/base';
      @import 'tailwindcss/components';
      @import 'tailwindcss/utilities';
      
      @layer base {
        :root {
          --color-primary-regular: 210 100% 50%;
          --color-primary-foreground: 0, 10%, 93%;

          --color-secondary-regular: 220 70% 20%;
          --color-secondary-foreground: 220 100% 98%;
          
          --color-neutral-black: 0 0% 7.5%;
