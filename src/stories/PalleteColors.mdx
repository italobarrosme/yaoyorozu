## Pattern installation styles

Install `postcss-import` and `tailwindcss` in the project.

<Unstyled>
  <Copy>
    {`
      npm install postcss-import tailwindcss
    `}
  </Copy>
</Unstyled>

The colors are defined in the root of the project in the file `tailwind.config.js` in the `colors` property.

<Unstyled>
  <Copy>
    {`
      module.exports = {
        colors: {
          primary: {
            regular: '#fda000',
            light: '#f2a516',
            dark: '#d95204',
          },
          secondary: {
            regular: '#121e1e',
            light: '#1d2b2b',
            dark: '#0c1616',
          },
          neutral: {
            black: '#000000',
            white: '#f9f9f9',
            light: '#c4c4c4',
            lightest: '#0000000a',
            dark: '#121e1e',
          },
        },
      };
    `}
  </Copy>
</Unstyled>

Create global styles for the colors and config tailwind in the file `*/global.css`.

<Unstyled>
  <Copy>
    {`
      @import 'tailwindcss/base';
      @import 'tailwindcss/components';
      @import 'tailwindcss/utilities';

      @import '@coqueirodigital/react-components/dist/styles.css';

      @layer base {
        :root {
          --color-primary-regular: #fda000;
          --color-primary-light: #F2A516;
          --color-primary-dark: #D95204;

          --color-secondary-regular: #121e1e;
          --color-secondary-light: #1d2b2b;
          --color-secondary-dark: #0c1616;

          --color-neutral-black: #000000;
          --color-neutral-white: #f9f9f9;
          --color-neutral-light: #c4c4c4;
          --color-neutral-lightest: #0000000a;
          --color-neutral-dark: #121e1e;

          --color-feedback-error: #f44336;
          --color-feedback-warning: #ff9800;
          --color-feedback-success: #4caf;
          --color-feedback-info: #2196f3;

          --color-complementary-highlight: #fd6600;

        }
      }
    `}
  </Copy>
</Unstyled>
